# AI Dashboard UI

A modern SaaS productivity dashboard built with React and TypeScript.

![Dashboard Preview](./public/preview.png)

## 🚀 Live Demo

[View Live Demo] [https://ai-dashboard-ui-liart.vercel.app/]

## ✨ Features

- 📊 Data visualization with interactive charts (line, bar, pie)
- ✅ Task management (add, complete, delete)
- 📝 Notes system (add, edit, delete)
- 🌙 Automatic dark mode via CSS custom properties
- 📱 Fully responsive design
- ⚡ Smooth animations with Framer Motion
- 🎨 Reusable component system (Card, Button, Badge)

## 🛠️ Tech Stack

- **React 19** → UI library
- **TypeScript** → Type safety
- **Vite** → Build tool
- **Zustand** → Global state management
- **Recharts** → Data visualization
- **Framer Motion** → Animations
- **CSS Custom Properties** → Theming and dark mode

## 📁 Project Structure

src/
components/
ui/          → Reusable components (Card, Button, Badge)
charts/      → Chart components (Line, Bar, Pie)
dashboard/   → Dashboard widgets (TaskWidget, NoteWidget)
ata/         → Mock data
layouts/     → Layout components
pages/       → Page components
store/       → Zustand store
types/       → TypeScript types

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/gbielbellamy/ai-dashboard-ui.git

# Install dependencies
npm install

# Start development server
npm run dev
```
